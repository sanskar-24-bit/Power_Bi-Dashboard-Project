Madhav Economics Sales Dashboard

This project is an interactive Sales Dashboard created using Power BI to analyze the sales performance of Madhav Economics. The dashboard provides meaningful business insights through data visualization and helps in tracking revenue, profit, customer trends, product performance, and regional sales analysis.

Key Features

- Sales and profit analysis
- Category and sub-category performance tracking
- Region-wise and state-wise sales insights
- Monthly and yearly trend analysis
- Interactive filters and slicers
- KPI cards for quick business overview

Tools & Technologies Used

- Power BI
- Excel / CSV Dataset
- Data Cleaning & Transformation
- DAX Measures
- Data Visualization

Project Objective

The main objective of this project is to transform raw sales data into actionable insights that support better business decision-making and performance monitoring.

Insights Generated

- Identified top-performing products and categories
- Analyzed sales trends over time
- Compared profit across different regions
- Tracked customer purchasing behavior

This dashboard demonstrates skills in data analysis, business intelligence, and dashboard development using Power BI.
